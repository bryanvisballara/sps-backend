Sube todo el contenido de esta carpeta a Hostinger para servirlo desde spste.com.

Variables a definir o actualizar en Render para el backend:

CORS_ORIGIN=https://spste.com,https://www.spste.com,http://localhost:5173,http://127.0.0.1:5173

Notas:
- No existe hoy una variable FRONTEND_URL en el backend.
- RENDER_BACKEND_URL puede quedarse apuntando a la URL actual del backend si no vas a mover la API fuera de Render.
- El frontend ya apunta por defecto a https://sps-backend-jxms.onrender.com/api cuando no se define VITE_API_URL.
- Si prefieres fijarlo explicitamente al volver a compilar el frontend, usa VITE_API_URL=https://sps-backend-jxms.onrender.com/api.
